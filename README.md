## **Miniprojeto Educacional (LGPD)**

# 

### **O que esse projeto faz?**



### • Conecta a um banco de dados

### • Busca dados da tabela usuarios

### • Oculta informações sensíveis (nome, CPF, e-mail, telefone)

### • Gera arquivos .csv com os dados

### • Mede o tempo de execução das funções

# 

### Para tornar este miniprojeto mais didático, foram acrescentados comentários junto ao código (isto facilitará a revisão/estudo futuro do referido trabalho).

# 

### **O que foi usado?**



### • Python

### • SQLAlchemy

### • PostgreSQL

### • CSV

### • Decorators (para medir tempo)
